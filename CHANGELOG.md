## [1.0.0] - 2024-05-10

### Added

- ✅ Telegram bot для управления задачами
- ✅ Интеграция с AI провайдерами (OpenRouter, Groq, OpenAI)
- ✅ Долгосрочная память через Mem0 AI
- ✅ Голосовая транскрипция (Whisper API)
- ✅ Управление задачами и расписанием
- ✅ Автоматические напоминания (cron-задачи)
- ✅ SQLite база данных с Drizzle ORM
- ✅ TypeScript для type-safety
- ✅ Полная документация (README, SETUP, ARCHITECTURE)

### Features

- 🤖 AI-ассистент с контекстом расписания

- 🎙️ Обработка голосовых сообщений

### Documentation

## Версионирование

Этот проект использует [Semantic Versioning](https://semver.org/):

- **MAJOR** версия для несовместимых изменений
- **MINOR** версия для новых функций (обратно совместимо)
- **PATCH** версия для исправлений ошибок (обратно совместимо)

Пример: `1.0.0` = MAJOR.MINOR.PATCH
