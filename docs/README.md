## 🚀 Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Umalanif/Schedule-bot.git
cd Schedule-bot

# 2. Установить зависимости
npm install

# 3. Создать .env файл
cp .env.example .env
# Отредактировать .env с вашими ключами

# 4. Инициализировать БД
npm run db:init

# 5. Запустить бот
npm start
```

## ❓ Часто задаваемые вопросы

**В:** Требуется ли оплата?  
**О:** Некоторые API (OpenAI) требуют оплату. OpenRouter и Groq предоставляют бесплатные кредиты при регистрации.

**В:** Где найти помощь?  
**О:** Создайте [issue](../../issues) или обсудите в [Discussions](../../discussions).

**В:** Как развернуть на сервер?  
**О:** Смотрите раздел "Развертывание на сервер" в [SETUP.md](../SETUP.md).

**В:** Можно ли использовать для нескольких пользователей?  
**О:** Текущая версия для одного пользователя. Поддержка нескольких пользователей планируется в версии 1.1.
